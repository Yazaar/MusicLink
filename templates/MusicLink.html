To use a custom service check out my chrome extension to create custom configurations for any website!

  1. Install my chrome extension here
  2. Set the SocketIO event to overlayUpdate
  3. Set the SocketIO host to https://yazaar.herokuapp.com
  4. Hit connect and make sure it connects successfully
  5. Select a room id above
  6. Add a browser source to your recording/streaming software with the generated URL, would recommend width 400 height 100 but modify to your preference
  7. Copy one of the imports underneath and paste it in the import preset field, then hit import (or create your own)
  8. Make sure the status is ON and go to the tab you would like to capture, then hit inject current tab (keep auto inject OFF)
